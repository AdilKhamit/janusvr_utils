1. If you want to export lightmaps, make sure your scene is using only Baked GI, not Precomputed Realtime GI.



2. Import the JanusVR Package by going on "Assets/Import Package/Custom Package"




3. Open the exporter by hitting Window -> Janus VR Exporter



4. Exporter Options


Option Description
Export Path The folder were exporting the scene to
Mesh format The format were going to export all meshes to (only FBX right now).
Uniform Scale A uniform scale to transform the entire scene.
Export GIFs If the export should export GIFs. If false, the GIFs get reprocessed to the texture format and only the first frame is exported.
Texture Format The format to export all textures. Supported formats are PNG and JPG. If the object needs alpha information inside the texture the format will be overriden to PNG always.
Texture Quality: Only available when using lossy texture formats, like JPG.
Export Materials If the exporter should output the materials (if disabled, lightmaps are still exported, so you can take a look at only lightmap data with a gray tone)
Export Skybox Exports the scene's skybox
Lightmap Type The type of lightmaps you want to export
Max Lightmap Resolution The maximum resolution a lightmap atlas can have

4.1. Export HTML only

This button is intended for when you have already exported the entire scene, but made a change that doesnt impact the already exported models/textures (like changing the position of an object in the scene).


4.2. Reset Parameters

Resets all parameters to their default values.


4.3. Full Export

Exports the entire scene - models, textures and the html file - to the output directory.


4. Lightmap Options
Option Description
None No lightmaps are exported.
Packed Packed into multiple atlases.
Packed Source EXR The optimal type of lightmaps: everything is exported straight out from Unity, packed into multiple atlases with HDR support.
Unpacked Unpacks the atlases into smaller textures, so no objects share the same lightmap texture. Consumes more memory with no gain. (Exists only for compatibility/debugging purposes)
Baked Bakes the lightmap into the object material. Consumes too much memory while making the main material look poor. (Exists only for compatibility/debugging purposes)

4. Components
Option Description
GameObject -> JanusVR -> Room Entry Portal Makes an entry portal on your Unity scene. You can position the portal and move it around, and this is going to be the entrance to your JanusVR room.
GameObject -> JanusVR -> Link Makes a link portal on your Unity scene. You can set up the URL, title and all other options, and this is going to be exported as a Link portal to JanusVR.
