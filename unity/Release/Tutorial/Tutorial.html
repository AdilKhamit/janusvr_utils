1. If you want to export lightmaps, make sure your scene is using only Baked GI, not Precomputed Realtime GI.



2. Import the JanusVR Package by going on "Assets/Import Package/Custom Package"




3. Exporter Options


Option Description
Export Path The folder were exporting the scene to
Default Mesh format The format to default every mesh to export to. Only FBX is supported right now, but OBJ support is coming.
Default Texture Format The format to default every texture to export to. PNG are lossless, so quality options are ignored. Supported formats are PNG and JPG.
Default Textures Quality The quality to initialize every texture that's using a lossy export format (like JPG).
Texture Filter The filtering mode to use when downsampling textures.
Compress Models Compress the scene models using GZip (WIP and extremely slow)
Uniform Scale A uniform scale to transform the entire scene.
Export Materials If the exporter should output the materials (if disabled, lightmaps are still exported, so you can take a look at only lightmap data with a gray tone)
Export Skybox Exports the scene's skybox, if it's a 6-sided skybox
Lightmap Type The type of lightmaps you want to export
Max Lightmap Resolution The maximum resolution a lightmap atlas can have

3.1. Export HTML only

This button is intended for when you have already exported the entire scene, but made a change that doesnt impact the already exported models/textures (like changing the position of an object in the scene).


3.2. Start Export

This is used to look up the entire scene and cache what is going to be exported on the next step.

Option Description
View Distance The farplane distance. This is calculated from the bounds of your scene, so changing it is inadvisable.
Per-Texture options The farplane distance. This is calculated from the bounds of your scene, so changing it is inadvisable.

4. Lightmap Options
Option Description
None No lightmaps are exported.
Packed The optimal type of lightmaps: everything is exported straight out from Unity, packed into multiple atlases.
Unpacked Unpacks the atlases into smaller textures, so no objects share the same lightmap texture. Consumes more memory with no gain. (Exists only for compatibility/debugging purposes)
Baked Bakes the lightmap into the object material. Consumes too much memory while making the main material look poor. (Exists only for compatibility/debugging purposes)
